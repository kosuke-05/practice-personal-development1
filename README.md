## システム概要
- ログインユーザーごとに登録したタスクを管理できるタスク管理アプリ
- 認証に成功したユーザーが自分の登録したタスクを閲覧・編集・削除することが出来る

## 主な機能
- ユーザー認証（ログイン / ログアウト）
- タスクのCRUD機能
- タスク検索（タスク名で部分一致）
- ステータスフィルター機能

## 使用技術
- Git
- GitHub
- Javascript
- Typescript
- React.js
- Next.js
- 状態管理：Zastand
- 仮サーバー：MSW

## 各ページ情報
一覧画面
- 全タスクの表示
- タスク名で検索する検索欄
- ステータスでタスクを絞り込むフィルター機能

新規作成画面
- 新規タスク登録時に遷移する

タスク編集画面
- 特定のタスク情報を編集する際に遷移する
- 編集対象のタスクはid番号で特定する

ユーザー認証画面
- ユーザーのメールアドレスとパスワードで認証を行う
